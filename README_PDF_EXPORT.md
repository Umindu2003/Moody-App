## Overview

The PDF export feature allows users to generate comprehensive mood reports in PDF format for different time periods (Today, Week, Month, Year). Users can download and share these reports with healthcare providers, therapists, or for personal record-keeping.

## How to Use PDF Export

### 1. Navigate to Stats Page

- Open the Moody app
- Tap on the "Stats" tab in the bottom navigation

### 2. Select Time Period

Choose your desired time period using the period selector buttons:

- **Today**: Current day's mood entries
- **Week**: Past 7 days of mood entries
- **Month**: Past 30 days of mood entries
- **Year**: Past 365 days of mood entries

### 3. Export PDF Report

- Tap the **"Export"** button (download icon) in the top right
- Confirm the time period you want to export
- The app will generate a comprehensive PDF report
- Choose to save to device or share with others

## Technical Features

### Security & Privacy

- All PDF generation happens locally on your device
- No data is sent to external servers
- Reports can be shared only when you choose to

### File Management

- PDFs are saved with clear naming: `Moody_Report_[period]_[date].pdf`
- Android: Automatically saved to "Moody Reports" album in gallery
- iOS: Available through standard sharing options
- Can be shared via email, cloud storage, messaging apps

### Permissions Required

- **Storage Access**: To save PDF files to your device
- **Media Library Access**: To organize reports in a dedicated album (Android)

## Troubleshooting

### "No Data Available" Error

- **Cause**: No mood entries exist for the selected time period
- **Solution**: Select a different time period or add mood entries first

### "Export Failed" Error

- **Cause**: Device permissions not granted or storage issues
- **Solutions**:
  1. Check app permissions in device settings
  2. Ensure sufficient storage space
  3. Restart the app and try again
  4. Update to latest app version

### PDF Not Generating

- **Cause**: Missing dependencies or configuration issues
- **Solutions**:
  1. Force close and reopen the app
  2. Check internet connection (may be needed for initial setup)
  3. Clear app cache (Android) or offload app (iOS)
